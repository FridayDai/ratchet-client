ratchet-v2-provider-desktop
===========================
## Dependencies

 1. JDK 1.7+
 2. Grails 2.4.4

 - Install **gvm**  Go to [gvm home page](http://gvmtool.net/)

 - Install **grails**
    ```bash
       $ gvm install grails
    ```
    
## Run
 - Start
    ```bash
       $ gvm install grails
    ```
